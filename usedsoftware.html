What software do I use?

Pretty much all of the software that I use is free/libre software, though there is a couple exceptions for school due to COVID (namely Google Classroom). Here is a non-comprehensive list of the software I use:

  • Operating System: Trisquel GNU/Linux-libre
  • Desktop Environment: MATE
  • Web Browser: GNU IceCat
  • Office Suite: LibreOffice
  • Text Editor: Nano (CLI)
  • Image Editor: Krita
  • Screen Recorder: OBS Studio
  • Media Player: MPV Media Player
  • C Compiler: GCC
