What software do I use?

A large portion of the software that I use is free/libre software, though there are unfortunately a few exceptions (namely Google Classroom and Steam). Here is a non-comprehensive list of the software I use:

Main Desktop (Ryzen 3, GTX 1050Ti):

  • Operating System: Artix GNU/Linux (runit init system)
  • Desktop Environment: MATE
  • Web Browser: Mozilla Firefox
  • Office Suite: LibreOffice
  • Text Editor: Nano (CLI) / Pluma (GUI)
  • Image Editor: Krita
  • Screen Recorder: OBS Studio
  • Media Player: Celluloid
  • C Compiler: GCC

Laptop (ASUS Chromebook C201P, Librebooted, USB WiFi):

  • Operating System: Debian GNU/Linux (PrawnOS, deblobbed kernel)
  • Desktop Environment: MATE
  • Web Browser: GNU Icecat
  • Office Suite: LibreOffice
  • Text Editor: Nano (CLI) / Pluma (GUI)
  • Image Editor: GIMP
  • Screen Recorder: SimpleScreenRecorder
  • Media Player: Celluloid
  • C Compiler: GCC
